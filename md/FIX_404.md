# 404 오류 해결 가이드

## 문제
- `/login` 페이지에서 404 오류
- `/auth/kakao/callback` 페이지에서 404 오류

## 해결 방법

### 1. Next.js 개발 서버 재시작

터미널에서:
```bash
# 현재 실행 중인 서버 중지 (Ctrl+C)
# 그 다음 다시 시작
npm run dev
```

### 2. 빌드 캐시 삭제 후 재시작

```bash
# .next 폴더 삭제
Remove-Item -Recurse -Force .next

# 개발 서버 재시작
npm run dev
```

### 3. 브라우저 캐시 삭제

- Chrome/Edge: `Ctrl + Shift + Delete` → 캐시된 이미지 및 파일 삭제
- 또는 시크릿 모드로 접속: `Ctrl + Shift + N`

### 4. 직접 URL 접근 확인

브라우저에서 직접 접근:
- `http://localhost:3000/login`
- `http://localhost:3000/auth/kakao/callback`

### 5. 파일 구조 확인

다음 파일들이 올바른 위치에 있는지 확인:
```
src/app/login/page.tsx ✅
src/app/auth/kakao/callback/page.tsx ✅
src/components/KakaoLoginButton.tsx ✅
```

### 6. 터미널 오류 확인

개발 서버를 실행한 터미널에서 오류 메시지가 있는지 확인하세요.

## 여전히 문제가 있다면

1. **포트 확인**: 다른 포트에서 실행 중인지 확인
2. **파일 저장**: 모든 파일이 저장되었는지 확인
3. **타입스크립트 오류**: `npm run build`로 빌드 오류 확인
